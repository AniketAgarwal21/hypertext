## Deployment

To run this project on your local machine, follow the steps:

- Clone the repository :

```bash
  git clone git@github.com:AniketAgarwal21/hypertext.git
```

- Get inside the project folder :

```bash
  cd hypertext
```

- Install the required dependencies :

```bash
  npm install
```

- Start the react app :

```bash
  npm start
```
